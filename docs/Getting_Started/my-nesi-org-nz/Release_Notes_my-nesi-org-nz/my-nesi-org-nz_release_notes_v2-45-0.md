## New and Improved
- Principal Investigator Role: Introduced the Principal Investigator role in the project application.
- Enhanced Validation: Added validation for email and phone number fields in the project application for both supervisors and principal investigators, ensuring data accuracy.
- Core UI Upgrade: Upgraded the core user interface to the latest framework. The application now features the REANNZ colour scheme for a consistent look.
- Holiday Banner: Prepared a holiday banner that will be displayed a few days before the end-of-year break.
- Relevant Storage Dates: Default start and end dates for long-term storage are now more relevant.
- Improved Help Information: Added guidance on the supported format for long-term storage names to assist users.
- Removal of outdated information: the Related information section was removed.

## Fixed
- Storage Units: The default value for long-term storage is now displayed in TiB (Tebibytes) instead of GiB (Gibibytes).
