## New and Improved
- Improved the end date calculation for new allocation requests so that it aligns more closely with the nearest renewal cycle.
- Updated dependencies to improve stability, security, and performance.
- Improved the wording on the Join Project page to make the process clearer and easier for users.
- On the Project Request page, users can no longer add themselves as a supervisor.

## Fixed
- It is now possible to add members to inactive projects when required.
- Projects won't be visible in my.nesi.org.nz if a Research Developer Cloud user account is disabled.
